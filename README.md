# AI Data Center Site Selection Engine

An interactive decision-support platform designed to evaluate AI data center deployment readiness across 50 countries.

The dashboard integrates renewable energy availability, electricity costs, climate conditions, and AI demand indicators to benchmark infrastructure competitiveness and identify strategic deployment opportunities.

Built using Tableau, Python, and SQL.

## Key Features

- Evaluated AI data center deployment readiness across 50 countries
- Built interactive Tableau dashboards with cross-filter drill-down functionality
- Compared country-specific metrics against global benchmarks
- Integrated multi-source datasets using Python and SQL
- Generated Top-10 deployment candidate rankings
